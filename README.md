# Project 3: Spots

An image sharing Website

### Overview

- HTML
- Figma
- CSS
- Responsive design

## Description

Spots is an interactive website where people can upload photos, it has profile customization with both a profile name and description and it has responsive design changing the layout from devices with smaller screens
